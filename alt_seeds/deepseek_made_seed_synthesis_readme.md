## üåü Overview

The **Recursive Steel Man Collider (RSMC)** is a meta-cognitive framework designed to analyze complex domains through recursive self-improvement. It systematically builds the strongest possible versions of competing positions, identifies irreducible disagreements, and connects analyses to concrete decisions‚Äîall while maintaining ethical alignment and managing complexity.

Born from 8 evolutionary cycles of self-application, v2.0 represents the synthesis of multiple framework versions into a unified, maximally-capable system.

## üé® Projection Planes

The framework analyzes every domain through four interconnected lenses:

### 1. **Mechanistic Plane**
- *Focus*: Causal structure, mechanisms, dependencies
- *Output*: Causal graphs, structural invariants
- *Question answered*: "How does it work?"

### 2. **Normative Plane**
- *Focus*: Values, ethics, stakeholder impacts
- *Output*: Value trade-offs, ethical constraints
- *Question answered*: "What should we value?"

### 3. **Epistemic Plane**
- *Focus*: Knowledge, uncertainty, evidence
- *Output*: Confidence intervals, evidence weights
- *Question answered*: "How confident can we be?"

### 4. **Decision-Theoretic Plane**
- *Focus*: Actions, outcomes, expected value
- *Output*: Policy recommendations, regret analysis
- *Question answered*: "What should we do?"
